# Доброе дело ✨
### Суть проекта:
Реализация консольной игры «Доброе дело» на языке C++, в которой ученики встают в круг и совершают добрые дела. Программа моделирует математическую игру с использованием собственной реализации двусвязного циклического списка. В процессе игры случайным образом выбирается ученик, который совершает «доброе дело» и увеличивает свой рейтинг.
### Ключевой функционал
- Загрузка списка учеников из текстового файла (начальный рейтинг равен 0).
- Генерация случайного числа из диапазона [-10; 10] для определения направления и количества шагов.
- Перемещение по кругу с использованием двусвязного циклического списка (собственная реализация без STL-контейнеров).
- Учёт рейтинга каждого ученика с выводом протокола каждого раунда.
- Сортировка результатов по убыванию рейтинга с сохранением устойчивости порядка.
- Обработка ошибок с использованием собственных классов исключений
